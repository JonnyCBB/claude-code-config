# eng-utils

Engineering utilities for incident investigation, architecture documentation, and service diagnostics.

## Installation

```bash
claude plugins add /path/to/eng-utils
```

## Skills (3)

| Skill                   | Command                    | Description                                                                       |
| ----------------------- | -------------------------- | --------------------------------------------------------------------------------- |
| refactor-architecture   | `/refactor-architecture`   | Explore codebase for architectural friction, quantify issues, design alternatives |
| system-architecture-doc | `/system-architecture-doc` | Create C4 architecture documentation with Structurizr DSL models                  |
| c4-architecture         | `/c4-architecture`         | C4 model architecture diagrams with Structurizr DSL                               |
